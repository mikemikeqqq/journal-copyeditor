# Journal Style Profiles

Use this reference when the target journal, citation style, or English variety matters.

## Selection Rule

If the user names a journal, follow that journal's current author instructions when supplied. If the instructions are not supplied, use the closest profile below and flag uncertain items in the report.

If no target journal is provided, default to:

- US English for general international journals;
- author-date citation checking for humanities/social science manuscripts;
- numeric citation checking for biomedical, Nature-style, IEEE, or Vancouver manuscripts when the manuscript already uses numbers.

## APA Author-Date

- In-text format: `(Author, Year)` or `Author (Year)`.
- Multiple citations in one parenthetical cluster are usually alphabetized by author in APA, not sorted by year.
- Reference list is alphabetized by first author, then year.
- Sentence case for article titles.
- DOI should use URL form when available: `https://doi.org/...`.

When the user specifically requests citations sorted by year, apply year sorting inside clusters and note the deviation if APA compliance is required.

## Harvard Author-Date

- In-text format: `(Author Year)` or `(Author, Year)`, depending on venue.
- Multiple citations are often ordered chronologically or alphabetically depending on house style.
- Reference list is usually alphabetized by first author.
- Check punctuation around year, page numbers, and `et al.` carefully.

Default for this skill: sort multiple citations in the same cluster by publication year ascending unless the journal says otherwise.

## Nature / Nature Portfolio

- Main research articles commonly use numbered references, not author-date clusters.
- References are numbered in order of first appearance.
- Do not sort numbered citations by year.
- Titles, journal abbreviations, volume, pages, and year must follow the journal's reference format.
- British spelling is often acceptable, but follow the journal or manuscript's existing variety.

Use citation-year sorting only if the manuscript is in an author-date format by journal instruction.

## Elsevier

- Elsevier journals vary: many use numbered Vancouver-like styles, some use Harvard author-date.
- Follow the target journal guide over a generic Elsevier rule.
- For author-date Elsevier styles, citation clusters are often chronological, then alphabetical for same year.
- For numbered styles, references are either order of citation or alphabetical depending on journal.

Always report the assumed style profile.

## IEEE

- Numeric bracket citations, usually `[1]`, `[2]`.
- References are numbered in order of first citation.
- Do not apply year sorting.
- Check whether each numeric citation has a matching numbered reference and whether numbers appear in ascending order when appropriate.

## Vancouver / ICMJE

- Numeric citations, usually in order of first appearance.
- References are numbered in citation order.
- Article titles are sentence case.
- Journal titles often use NLM abbreviations.
- DOI is useful but journal requirements vary.

Do not apply author-date sorting.

## English Variety

Default to preserving the manuscript's existing variety unless the user or journal specifies another.

Common choices:

- US: analyze, behavior, modeling, tumor, randomized.
- UK: analyse, behaviour, modelling, tumour, randomised.

Do not standardize quoted titles, journal names, references, dataset names, software names, or proper nouns.

## Title and Heading Style

- APA and many social science journals: title case may apply to manuscript title, sentence case to reference article titles.
- Nature-style scientific prose: concise titles, avoid inflated claims.
- IEEE/engineering: preserve technical specificity and search terms.

When uncertain, flag the style decision in the report rather than silently imposing a house style.
